🚀 Monta ↔ Odoo Integration Module

A seamless bridge between Odoo and Monta NL for automated order handling

The Monta-Odoo Integration module enables your Odoo ERP to communicate directly with Monta NL’s API (v6), automating order synchronization, tracking updates, and delivery status management — all without manual effort.

Designed for eCommerce businesses that rely on Monta as their fulfillment partner, this integration ensures that every order in Monta is synced securely, quickly, and reliably with your Odoo instance.

📦 Module Overview

Once installed, the module connects to Monta via REST API and allows you to fetch orders, track deliveries, and view Monta order data directly inside Odoo. The system intelligently prevents duplicate orders and maintains a log for every sync operation.

📊 Monta Order Dashboard

The Monta Order Dashboard provides clear visibility into all synchronized orders. You can see Monta references, tracking information, and delivery statuses directly within Odoo’s Sales Orders view. The dashboard supports filtering, search, and detailed status monitoring.

⚙️ Configuration Steps

Configuration is quick and secure. All credentials are stored safely using Odoo’s ir.config_parameter system. To set up the connection:

  1. Go to Settings → Technical → System Parameters
  2. Add the following parameters:
  • monta.username → Your Monta API username
  • monta.password → Your Monta API password
  • monta.base_url → Monta API Base URL (e.g. https://api-v6.monta.nl/)
  • monta.timeout → Timeout value (in seconds)

Once configured, the system will automatically connect to Monta and begin syncing orders based on your setup preferences.

💡 Features

  • ✔️ Fetch orders directly from Monta NL into Odoo
  • ✔️ Avoid duplicate orders through intelligent sync logic
  • ✔️ Real-time tracking number and delivery status updates
  • ✔️ Mirror fields like Monta Status, Track & Trace, and Source
  • ✔️ Securely handle credentials using system parameters
  • ✔️ Robust logging for all API interactions

🔒 Security & Logging

Each API call (both request and response) is logged securely in the backend for monitoring and debugging purposes. Sensitive information like passwords is never exposed or stored in plain text.

Logs include timestamps, order references, and response statuses from Monta to provide complete traceability and make troubleshooting easy in production environments.

🧰 Technical Details

  • ✔️ API Version: Monta v6 (REST)
  • ✔️ Authentication: Basic Auth
  • ✔️ Supported Odoo Versions: 15.0 – 17.0
  • ✔️ Logging: Built-in Python logging with Odoo integration
  • ✔️ Auto-retry on API timeout

🧩 Use Cases

  • Automate Monta order imports for your online store
  • Provide warehouse teams instant access to fulfillment data
  • Monitor order delivery statuses directly inside Odoo
  • Ensure all customer orders from Monta are synced, clean, and traceable

🛠 Troubleshooting

If connection fails:

  • Double-check your monta.base_url — it should end with a slash (/).
  • Ensure username and password are correct and active on Monta API portal.
  • Verify that the timeout value isn’t too low for larger data sets.
  • Review Odoo logs for any API error codes or response messages.
